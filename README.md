App Features

1. Product Homepage

Displays product categories at the top
Search bar for quick product lookup
Scrollable product list with:
- Product image
- Current price
- New price per unit
- Wholesale price
- Quantity selector

2. Product Details & Cart Interaction

Each product card includes:
- Add button (+)
- Compact price breakdown
- Quantity dropdown (e.g., 1–10 units)
- Users can add multiple items easily before checkout

3. Order Summary Page

Displays all selected products
Shows:
- Product name
- Selected quantity
- Subtotal price per product
- Auto-calculates total payment
- “Checkout” button to proceed with purchase

4. User Roles

The system supports three roles:
- Farmer : Upload products, set prices, manage stock
- Wholesaler : Bulk purchasing features
- Buyer : Purchase pineapple products individually

5. Product Management

Farmers can:
- Add new pineapple products
- Set current, new, and wholesale prices
- Update inventory
- Products are instantly reflected in buyer/wholesaler views

6. Smooth Navigation

Simple bottom navigation:
- Home (Product list)
- Cart (Checkout page)
- Profile (Account management)

7. Seamless Checkout

Checkout screen summarises:
- Selected items
- Total quantity
- Total price
- User proceeds with order placement 

8. Order Tracking 

- Order request goes to farmer
- Farmer can accept / prepare order
- Buyer sees updated order status
